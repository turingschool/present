### Instructions for running locally

* Clone
* Bundle
* Go to Google Cloud Console, find client_id and secret
* Run Figaro install
* Add GOOGLE_OAUTH_CLIENT_ID and GOOGLE_OAUTH_CLIENT_SECRET to application.yml
